---
title: 'Create Post'
api: 'POST https://api.infinipost.co/v1/posts'
description: 'Create and schedule a video or slideshow post'
---

## Request Body

<ParamField body="accountIds" type="array" required>
  Array of account IDs to post to (e.g., `["acc_123", "acc_456"]`).
</ParamField>

<ParamField body="contentType" type="string" required>
  Type of content: `video` or `slideshow`.
</ParamField>

<ParamField body="contentUrl" type="string | array" required>
  - For `video`: Single URL string
  - For `slideshow`: Array of image URLs
</ParamField>

<ParamField body="caption" type="string">
  Post caption/description.
</ParamField>

<ParamField body="soundUrl" type="string">
  URL of audio/music file to use (optional).
</ParamField>

<ParamField body="scheduledTime" type="string">
  ISO 8601 timestamp for when to post. If not provided, posts immediately.
</ParamField>

<ParamField body="slideDuration" type="number" default="4">
  For slideshows only: Duration per slide in seconds (4, 6, or 8).
</ParamField>

## Response

<ResponseField name="posts" type="array">
  Array of created post objects (one per account).
</ResponseField>

<RequestExample>

```bash Video Post
curl -X POST https://api.infinipost.co/v1/posts \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "accountIds": ["acc_123", "acc_456"],
    "contentType": "video",
    "contentUrl": "https://example.com/video.mp4",
    "caption": "Check out this amazing content! #viral",
    "soundUrl": "https://example.com/audio.mp3",
    "scheduledTime": "2024-04-22T12:00:00Z"
  }'
```

```bash Slideshow Post
curl -X POST https://api.infinipost.co/v1/posts \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "accountIds": ["acc_123"],
    "contentType": "slideshow",
    "contentUrl": [
      "https://example.com/slide1.jpg",
      "https://example.com/slide2.jpg",
      "https://example.com/slide3.jpg"
    ],
    "caption": "Swipe for more! 👉",
    "slideDuration": 4
  }'
```

</RequestExample>
