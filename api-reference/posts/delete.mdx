---
title: 'Delete Post'
api: 'DELETE https://api.infinipost.co/v1/posts/{id}'
description: 'Delete a post from TikTok or Instagram'
---

## Path Parameters

<ParamField path="id" type="string" required>
  The unique post identifier.
</ParamField>

## Response

<ResponseField name="id" type="string">
  The deleted post's identifier.
</ResponseField>

<ResponseField name="deleted" type="boolean">
  Always `true` if deletion was successful.
</ResponseField>

<ResponseField name="deletedAt" type="string">
  ISO 8601 timestamp of deletion.
</ResponseField>

<RequestExample>

```bash cURL
curl -X DELETE https://api.infinipost.co/v1/posts/post_abc123 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://api.infinipost.co/v1/posts/post_abc123', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});

const data = await response.json();
```

</RequestExample>

<ResponseExample>

```json Response
{
  "id": "post_abc123",
  "deleted": true,
  "deletedAt": "2024-04-21T20:20:00Z"
}
```

</ResponseExample>

<Warning>
  Scheduled posts can be deleted before posting. Posted content will be removed from the platform.
</Warning>
