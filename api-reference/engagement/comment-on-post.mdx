---
title: 'Comment on Post'
api: 'POST https://api.infinipost.co/v1/posts/{id}/comments'
description: 'Add a comment to a post'
---

## Path Parameters

<ParamField path="id" type="string" required>
  The post ID to comment on (can be your own post or another user's).
</ParamField>

## Request Body

<ParamField body="accountId" type="string" required>
  The account ID to comment from.
</ParamField>

<ParamField body="text" type="string" required>
  The comment text.
</ParamField>

## Response

<ResponseField name="id" type="string">
  Unique comment identifier.
</ResponseField>

<ResponseField name="postId" type="string">
  The post that was commented on.
</ResponseField>

<ResponseField name="accountId" type="string">
  Account that created the comment.
</ResponseField>

<ResponseField name="text" type="string">
  Comment text.
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp.
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST https://api.infinipost.co/v1/posts/post_abc123/comments \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "accountId": "acc_123",
    "text": "Amazing content! 🔥"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.infinipost.co/v1/posts/post_abc123/comments', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    accountId: 'acc_123',
    text: 'Amazing content! 🔥'
  })
});

const data = await response.json();
```

</RequestExample>
