---
title: 'Configure Warmup'
api: 'PATCH https://api.infinipost.co/v1/accounts/{id}/warmup/configure'
description: 'Update warmup settings for an active warmup'
---

## Path Parameters

<ParamField path="id" type="string" required>
  The unique account identifier.
</ParamField>

## Request Body

<ParamField body="keywords" type="array">
  Update keywords for content discovery.
</ParamField>

<ParamField body="profilesToFollow" type="array">
  Update list of profiles to follow.
</ParamField>

<ParamField body="postsPerDay" type="number">
  Update number of posts per day (1-5).
</ParamField>

## Response

<ResponseField name="accountId" type="string">
  The account identifier.
</ResponseField>

<ResponseField name="warmupStatus" type="object">
  Updated warmup configuration.
</ResponseField>

<RequestExample>

```bash cURL
curl -X PATCH https://api.infinipost.co/v1/accounts/acc_1a2b3c4d5e/warmup/configure \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "keywords": ["streetwear", "sneakers", "hypebeast"],
    "postsPerDay": 4
  }'
```

```python Python
import requests

response = requests.patch(
    'https://api.infinipost.co/v1/accounts/acc_1a2b3c4d5e/warmup/configure',

        'Authorization': 'Bearer YOUR_API_KEY',

    },
```
