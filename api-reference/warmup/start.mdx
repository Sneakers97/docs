---
title: 'Start Warmup'
api: 'POST https://api.infinipost.co/v1/accounts/{id}/warmup/start'
description: 'Begin the automated 7-day warmup process'
---

## Path Parameters

<ParamField path="id" type="string" required>
  The unique account identifier.
</ParamField>

## Request Body

<ParamField body="keywords" type="array">
  Array of keywords to use for content discovery during warmup (e.g., `["fashion", "style", "outfit"]`).
  
  Default: Platform-appropriate trending keywords.
</ParamField>

<ParamField body="profilesToFollow" type="array">
  Array of usernames to follow during warmup (e.g., `["fashionblogger", "styleinspo"]`).
  
  Optional: If not provided, accounts will be auto-selected based on keywords.
</ParamField>

<ParamField body="postsPerDay" type="number" default="3">
  Number of posts to publish per day during warmup (1-5).
</ParamField>

## Response

<ResponseField name="accountId" type="string">
  The account identifier.
</ResponseField>

<ResponseField name="warmupStatus" type="object">
  Warmup configuration and status.
  
  <Expandable title="properties">
    <ResponseField name="enabled" type="boolean">
      Always `true` after starting warmup.
    </ResponseField>
    
    <ResponseField name="currentDay" type="number">
      Current day of warmup (1-7).
    </ResponseField>
    
    <ResponseField name="startedAt" type="string">
      ISO 8601 timestamp of when warmup started.
    </ResponseField>
    
    <ResponseField name="keywords" type="array">
      Keywords being used.
    </ResponseField>
    
    <ResponseField name="postsPerDay" type="number">
      Posts per day configuration.
    </ResponseField>
  </Expandable>
</ResponseField>

<Info>
  **What Happens During Warmup?**
  
  - **Day 1-2:** Light engagement (views, likes)
  - **Day 3-4:** Follows and comments begin
  - **Day 5:** TikTok username can be set
  - **Day 6-7:** Full engagement patterns
  
  This gradual ramp-up mimics authentic user behavior and helps avoid platform restrictions.
</Info>
