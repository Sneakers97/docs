---
title: 'List Accounts'
api: 'GET https://api.infinipost.co/v1/accounts'
description: 'Retrieve a list of all your accounts'
---

## Query Parameters

<ParamField query="platform" type="string">
  Filter by platform: `tiktok` or `instagram`.
</ParamField>

<ParamField query="status" type="string">
  Filter by status: `active`, `warmup`, `pending`, or `failed`.
</ParamField>

<ParamField query="limit" type="number" default="50">
  Number of accounts to return (max 100).
</ParamField>

<ParamField query="offset" type="number" default="0">
  Number of accounts to skip for pagination.
</ParamField>

## Response

<ResponseField name="data" type="array">
  Array of account objects.
</ResponseField>

<ResponseField name="total" type="number">
  Total number of accounts matching the filters.
</ResponseField>

<ResponseField name="hasMore" type="boolean">
  Whether there are more accounts beyond this page.
</ResponseField>

<RequestExample>

```bash cURL
curl https://api.infinipost.co/v1/accounts?platform=tiktok&limit=10 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://api.infinipost.co/v1/accounts?platform=tiktok&limit=10', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.infinipost.co/v1/accounts',
    headers={'Authorization': 'Bearer YOUR_API_KEY'},
    params={'platform': 'tiktok', 'limit': 10}
)

print(response.json())
```
